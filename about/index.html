About

Hi, I'm Centurybbx. I'm a developer who enjoys building things that live on the web.

My focus is on creating clean, functional interfaces that work well for everyone. I believe that good software is not just about code — it's about understanding the problem you're solving and the people you're solving it for.

What I'm Into

Building Games

I love creating browser-based games. There's something satisfying about turning an idea into something interactive that others can play.

Clean Code

I care about code that reads well and is easy to maintain. Not everything needs to be clever — clarity wins in the long run.

Learning

The web moves fast, and I enjoy staying curious. Whether it's a new framework, a different approach, or something entirely new.

Design Details

Small things matter. Proper spacing, consistent typography, and thoughtful interactions make experiences feel polished.

My Mindset

"Done is better than perfect" — but "perfect" is a moving target anyway. Ship early, iterate often, and keep learning.

I believe in shipping work that you're proud of, even if it's not perfect. The best software comes from continuous improvement, not endless planning.

Connect

Feel free to reach out if you want to chat about projects, ideas, or just say hi.
